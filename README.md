# Personal Finance Tracker (Demo)

Next.js personal finance tracker with:
- Demo login (username/password)
- Transaction CRUD (create/update/delete)
- Category management
- Monthly income/budget/savings targets
- Dashboard KPIs and charts
- Mock persistence via browser `localStorage` using Zustand store persistence

## Demo credentials

- Username: `demo`
- Password: `demo123`

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech

- Next.js (App Router, TypeScript)
- Zustand (state management + persisted store)
- Recharts (dashboard charts)
